# fake-news-detector 
Fake News Detector

A modern AI-powered Fake News Detection web application built using Object-Oriented Programming concepts and modern frontend technologies. This project helps users analyze news articles and determine whether the content is likely to be real or fake based on intelligent text analysis.

Developed as an academic project submission for the Object-Oriented Programming (OOP) subject by CSE Engineering students.

Features
AI-based fake news analysis
Interactive and responsive user interface
Risk score prediction system
Real-time text input analysis
Animated modern frontend design
Clean component-based architecture using OOP principles
Technologies Used
React.js
JavaScript
CSS
JSX Components
OOP Concepts
Project Structure
App.jsx → Main application component
AnalysisForm.jsx → User input and analysis form
RiskScoreCard.jsx → Displays prediction/risk score
HeroSection.jsx → Landing section UI
AnimatedBackground.jsx → Visual background effects
Footer.jsx → Footer component
Objective

The main objective of this project is to apply Object-Oriented Programming concepts in a real-world application while creating an intelligent system capable of identifying fake news content.
